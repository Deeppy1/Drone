3d Printed Drone Project


I'm currently attempting to create and 3d print a drone using an Esp32 and some old motors I had lying around

At the moment I'm sorting out motor controllers and connecting my Xbox controller


How I'm doing it

I'm using a libary called bluepad32 this allows me to connect a controller (i.e. xbox or playstation or a long list of others) and use that to set variables and control motors

Along with this I'm going to make the entire thing opensource from code to design files
